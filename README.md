#Dart ATM Simulator

Simulador simples de terminal de caixa eletrônico feito em linguagem Dart

## Funcionalidades

- Checar saldo
- Depositar dinheiro
- Sacar dinheiro
- Validações de Entrada

## Tecnologias

- Dart
- CLI

## Como usar

- Certifique-se de que o Dart está instalado.
- Run: dart run main.dart
